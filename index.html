자주 묻는 질문

Toolhub는 뭐하는 곳인가?

Toolhub는 단위가격 계산기처럼 일상에서 바로 쓸 수 있는 무료 웹 도구를 한 곳에 모아둔 온라인 계산기 모음입니다. 회원가입이나 앱 설치 없이 브라우저만 열면 바로 사용할 수 있고, 각 도구는 하나의 문제만 빠르게 해결하도록 만들어졌습니다.

도구는 전부 무료인가?

네, Toolhub 에 등재된 모든 웹 유틸리티 모음은 완전 무료입니다. 결제나 구독 없이 설치 없는 계산기를 그대로 사용하시면 됩니다. 별도 유료 기능이 생기면 도구 페이지에 명시적으로 안내합니다.

데이터가 서버로 전송되나?

아니오. Toolhub 에 등재된 도구들은 pure-static 방식으로 만들어져 입력값을 서버로 전송하지 않고 브라우저 안에서만 계산합니다. 편의를 위한 마지막 입력값 저장 등은 이 브라우저의 localStorage 에만 남고, 외부로 전송되지 않습니다.

새 도구는 얼마나 자주 나오나?

검증된 아이디어부터 순서대로, 주 2-3개 페이스로 새 도구를 만들어 이 페이지의 도구 모음에 추가합니다. 새 도구가 런칭되면 이 목록에 카드가 바로 추가되니 가끔 다시 들러 확인해 보세요.
